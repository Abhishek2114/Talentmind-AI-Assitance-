# **App Name**: TalentMind AI Assistant

## Core Features:

- Resume Parsing: Parse resumes to extract key information such as skills, experience, and education.
- Job Description Analysis: Analyze job descriptions to identify required skills and experience using AI tool.
- Skill Extraction: Extract skills from both resumes and job descriptions.
- Skill Gap Analysis: Identify skill gaps between a candidate's resume and a job description using AI tool.
- Job Recommendation: Recommend jobs to candidates based on their skills and experience. The recommendations use the embeddings generated by the provided scripts.
- GenAI Feedback: Generate feedback on a candidate's resume using generative AI to make improvements.
- Interactive Dashboard: Provide a user interface for candidates to view skill gaps, get resume feedback, and explore job recommendations.

## Style Guidelines:

- Primary color: Navy blue (#2E3192) for a professional and trustworthy feel.
- Background color: Light gray (#F0F2F5) to provide a clean and neutral backdrop.
- Accent color: Teal (#008080) to highlight important information and calls to action.
- Font pairing: 'Inter' (sans-serif) for body text, and 'Space Grotesk' (sans-serif) for headlines to convey a modern, technical feel.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use simple, professional icons to represent skills, job titles, and other relevant information.
- Subtle animations and transitions to improve user experience, such as loading indicators and smooth scrolling.
